    answer: using var can change its value and it can re-declared
            using let it cannot re-declared and it's value can changed
            using const cannot re-declared and cannot changed it's value.

    - 2️⃣ What is the spread operator (...)?

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    answer: map returns a modified array
        filter returns a new array with matched elements
        forEach method used to run code for each elements in ana array.
